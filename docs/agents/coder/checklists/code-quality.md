# Code Quality Standards

**Reference guide** for coding standards

**See**: Full details in `CLAUDE-old-backup.md` Section 2.2-2.4

## Quick Standards

### TypeScript
- No `any` types
- Export interfaces
- Strict mode enabled

### i18n
- ALL frontend text uses translation keys
- No hardcoded strings

### API
- REST conventions
- Input validation (Zod)
- Proper error handling

### Frontend
- Follow existing patterns
- Tailwind CSS
- TanStack Query for API calls
